## 🗂️ 文档使用建议

### 新用户
1. 先读 **README.md** 了解项目
2. 按照 **INSTALL.md** 安装
3. 参考 **USER_GUIDE.md** 学习使用

### 开发者
1. 先读 **README.md** 了解项目
2. 按照 **DEVELOPMENT.md** 搭建环境
3. 参考 **PROJECT_FINAL_SUMMARY.md** 了解架构
4. 阅读 **CONTRIBUTING.md** 了解贡献流程

### 测试人员
1. 参考 **INSTALL.md** 安装测试环境
2. 按照 **HOW_TO_TEST.md** 执行测试
3. 在 **USER_GUIDE.md** 中查找功能说明
