🚀 快速开始
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. 安装依赖:     pnpm install
  2. 构建项目:     pnpm run build
  3. 加载扩展:     chrome://extensions/ → 加载 dist 文件夹
  4. 开始使用:     Ctrl+Shift+P 打开侧边栏

✨ 核心亮点
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. 标签碎片化设计 - 不是简单的字符串，而是结构化的数据模型
  2. 多格式支持     - 同一 Prompt 可导出为 MJ/SD 不同格式
  3. 权重可视化     - 每个标签独立权重，直观显示
  4. 类型安全       - 完整的 TypeScript 类型定义
  5. 监控思维       - 数据模型预留版本追踪和效果评估接口
