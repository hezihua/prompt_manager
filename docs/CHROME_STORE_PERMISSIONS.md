# Chrome Web Store 权限说明文案

## 📋 复制粘贴使用

### 1. 单一用途说明（如果需要）

```
本扩展专注于帮助 AI 艺术家管理和优化 AI 绘图 Prompt，提供可视化编辑、版本控制和数据管理功能。所有功能都围绕 Prompt 管理这一核心目标。
```

---

### 2. contextMenus 权限说明

**权限名称**: `contextMenus`

**使用理由**:
```
用于在用户右键点击文本时，快速将选中的文本保存为 Prompt 片段，提升用户操作效率。
```

---

### 3. notifications 权限说明

**权限名称**: `notifications`

**使用理由**:
```
用于在 Prompt 保存成功、导入完成等操作后，向用户显示简短的通知提示，改善用户体验。
```

---

### 4. sidePanel 权限说明

**权限名称**: `sidePanel`

**使用理由**:
```
用于在浏览器侧边栏显示 Prompt 管理界面，方便用户在浏览网页的同时管理和编辑 AI 绘图 Prompt。
```

---

### 5. storage 权限说明

**权限名称**: `storage`

**使用理由**:
```
用于在用户浏览器本地存储 Prompt 数据、用户设置和版本历史记录。所有数据仅存储在本地，不上传到任何服务器。
```

---

### 6. 远程代码说明

**问题**: 是否使用远程代码？

**答案**: 否

**说明**:
```
本扩展不使用任何远程托管的代码。所有代码都打包在扩展文件中，不从外部服务器加载任何脚本。
```

---

### 7. 数据使用声明

**问题**: 扩展如何处理用户数据？

**答案**: 
```
本扩展不收集任何用户数据。所有 Prompt 数据和用户设置仅存储在用户浏览器本地（使用 Chrome Storage API），不会上传到任何服务器或与第三方共享。用户可以随时通过扩展的导出功能备份数据，或通过卸载扩展删除所有数据。
```

---
