## 🎯 核心功能

### 1. Prompt 管理
- **创建与编辑**: 可视化的标签编辑器
- **搜索与过滤**: 实时搜索、多维度过滤
- **组织管理**: 收藏、标签分类、排序
- **批量操作**: 批量删除、批量收藏

### 2. 标签系统
- **6种标签类型**: 主体、风格、光效、构图、技法、自定义
- **权重调节**: 0.5-2.0 可调节权重
- **拖拽排序**: 直观的拖拽重排
- **40+ 预设**: 常用标签库，一键添加

### 3. 版本控制
- **自动快照**: 每次保存自动创建版本
- **时间轴视图**: 可视化版本历史
- **版本对比**: 详细的差异分析
- **一键恢复**: 安全的版本回滚
- **图片关联**: 上传生成结果图
- **评分系统**: 5星评价版本质量

### 4. AI 参数
- **多模型支持**: Midjourney、Stable Diffusion、DALL-E
- **预设参数**: 常用参数快速添加
- **自定义参数**: 灵活的参数配置
- **智能提示**: 根据模型显示对应参数

### 5. 数据管理
- **完整备份**: JSON 格式导出所有数据
- **表格导出**: CSV 格式，Excel 可用
- **灵活导入**: 合并或替换模式
- **数据验证**: 防止错误导入

### 6. 用户体验
- **暗黑模式**: 3种主题（浅色/深色/自动）
- **响应式设计**: 适配不同屏幕
- **流畅动画**: 平滑的交互效果
- **键盘快捷键**: Ctrl+Shift+P 快速打开
