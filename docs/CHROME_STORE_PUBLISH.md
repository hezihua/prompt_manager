## 📦 步骤 1：准备扩展包

### 1.1 创建发布包

```bash
cd /home/hezihua/workspace/prompt_manager

# 清理并重新构建
rm -rf dist
pnpm run build

# 验证构建
ls -lah dist/
```

### 1.2 打包为 ZIP

Chrome Web Store 需要 ZIP 格式的扩展包：

```bash
# 创建 ZIP 包
cd dist
zip -r ../prompt-manager-v0.5.0.zip .

# 返回项目根目录
cd ..
```

**或者使用 GUI 工具**:
1. 打开文件管理器
2. 右键点击 `dist` 文件夹
3. 选择"压缩" / "创建压缩文件"
4. 命名为 `prompt-manager-v0.5.0.zip`

### 1.3 验证 ZIP 包

```bash
# 检查 ZIP 内容
unzip -l prompt-manager-v0.5.0.zip | head -20

# 验证必需文件
unzip -l prompt-manager-v0.5.0.zip | grep -E "manifest.json|service-worker.js|index.html|icons"
```

确保包含：
- ✅ manifest.json
- ✅ background/service-worker.js
- ✅ src/sidepanel/index.html
- ✅ src/popup/index.html
- ✅ icons/ 目录

## ⏱️ 审核时间

### 预计审核时长

- **首次提交**: 通常 3-7 个工作日
- **更新版本**: 通常 1-3 个工作日

### 审核状态

可以在 Developer Dashboard 查看审核状态：
- **待审核** (Pending Review)
- **审核中** (In Review)
- **已发布** (Published)
- **被拒绝** (Rejected) - 会说明原因

### 常见拒绝原因

1. **权限过度**: 申请了不必要的权限
   - 解决：移除不必要的权限

2. **功能不明确**: 描述不清楚
   - 解决：提供更详细的说明和截图

3. **隐私政策缺失**: 需要但未提供
   - 解决：添加隐私政策链接

4. **质量问题**: 功能不完整或有明显 Bug
   - 解决：修复问题后重新提交
