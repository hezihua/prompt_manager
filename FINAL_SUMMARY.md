## 🚀 如何开始使用

### 对于用户

1. **下载项目**
   ```bash
   git clone https://github.com/yourusername/prompt-manager.git
   cd prompt-manager
   ```

2. **安装依赖并构建**
   ```bash
   pnpm install
   pnpm run build
   ```

3. **在 Chrome 中加载**
   - 打开 `chrome://extensions/`
   - 开启"开发者模式"
   - 加载 `dist` 文件夹

4. **开始使用**
   - 按 `Ctrl+Shift+P` 打开侧边栏
   - 右键保存任意 Prompt
   - 享受高效的 Prompt 管理！

## 💡 核心价值主张

### 为什么要用这个工具？

#### 痛点 1: Prompt 管理混乱
**传统方式**: 记事本、便签、聊天记录
**我们的方案**: 结构化存储、标签分类、智能搜索

#### 痛点 2: 难以复用优质 Prompt
**传统方式**: 复制粘贴，容易丢失
**我们的方案**: 一键复制，格式自动转换

#### 痛点 3: 无法理解 Prompt 结构
**传统方式**: 看到什么存什么
**我们的方案**: 标签系统帮助理解每部分作用

#### 痛点 4: 隐私担忧
**传统方式**: 上传到第三方平台
**我们的方案**: 完全本地存储，不上传云端

### 与竞品的差异

| 功能 | AI Prompt 管理器 | 竞品 A | 竞品 B |
|------|-----------------|--------|--------|
| 标签系统 | ✅ 6种类型 | ❌ | ⚠️ 简单 |
| 权重管理 | ✅ 独立设置 | ❌ | ❌ |
| 本地存储 | ✅ | ✅ | ❌ |
| UI 美观度 | ⭐⭐⭐⭐⭐ | ⭐⭐ | ⭐⭐⭐ |
| 开源 | ✅ MIT | ❌ | ⚠️ 部分 |
| 多格式 | ✅ MJ/SD | ❌ | ⚠️ |
