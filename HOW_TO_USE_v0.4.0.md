# v0.4.0 测试指南

## 🔧 如何访问新功能

### 1. 重新加载扩展

由于你已经加载了旧版本，需要重新加载才能看到新功能：

1. 打开 Chrome 扩展页面：`chrome://extensions/`
2. 找到"AI Prompt 管理器"
3. 点击刷新按钮（🔄）
4. 关闭当前的侧边栏
5. 重新打开侧边栏

### 2. 找到设置按钮

重新加载后，你应该能在顶部看到：

```
标题："Prompt 管理器"
右侧按钮：
  - [选择] 按钮（批量操作）
  - [⚙️] 齿轮图标（设置）
  - [+ 新建] 按钮
```

### 3. 访问导出功能

1. 点击右上角的 **⚙️ 齿轮图标**
2. 设置面板会打开
3. 点击顶部的 **"数据管理"** 标签
4. 你会看到三个导出按钮：
   - **导出所有数据 (JSON)** - 完整备份
   - **导出项目列表 (CSV)** - Excel 格式
   - **导出版本快照 (CSV)** - 快照列表

## 📥 导出功能使用

### 导出所有数据（推荐用于备份）
1. 点击"导出所有数据 (JSON)"
2. 文件会自动下载，命名格式：`prompt-manager-backup-[时间戳].json`
3. 包含：所有项目、所有快照、所有设置

### 导出项目列表（用于 Excel 分析）
1. 点击"导出项目列表 (CSV)"
2. 文件会自动下载，命名格式：`prompt-manager-projects-[时间戳].csv`
3. 可用 Excel 或其他表格软件打开

### 导出版本快照
1. 点击"导出版本快照 (CSV)"
2. 文件会自动下载，命名格式：`prompt-manager-snapshots-[时间戳].csv`

## 📤 导入功能使用

### 两种导入模式

**合并模式（推荐）**：
- 保留现有数据
- 只添加新数据（ID 不重复的）
- 安全，不会丢失数据

**替换模式（谨慎）**：
- 清空所有现有数据
- 完全替换为导入的数据
- 有⚠️警告提示

### 导入步骤
1. 打开设置 → 数据管理
2. 选择导入模式（合并/替换）
3. 点击"选择 JSON 文件导入"
4. 选择之前导出的 JSON 文件
5. 查看导入统计
6. 页面会自动刷新

## 🎨 主题切换

### 三种主题模式

1. 打开设置 → 常规设置
2. 在"主题模式"区域选择：
   - **浅色** ☀️ - 传统白色界面
   - **深色** 🌙 - 黑色背景，保护眼睛
   - **自动** ⚡ - 跟随系统主题

3. 点击"保存设置"
4. 主题立即生效

## 🔲 批量操作

### 进入批量模式
1. 点击顶部的"选择"按钮
2. 界面变化：
   - 每个 Prompt 卡片出现复选框
   - 顶部出现批量操作按钮

### 批量操作步骤
1. **选择项目**：点击复选框或卡片
2. **全选**：点击"全选"按钮
3. **批量收藏**：点击"收藏 (N)"
4. **批量删除**：点击"删除 (N)"（有确认）
5. **取消**：点击"取消"退出选择模式

## ⚠️ 故障排查

### 如果看不到设置按钮

**方法1：清空缓存重新构建**
```bash
cd /home/hezihua/workspace/prompt_manager
rm -rf dist
pnpm run build
```

**方法2：完全重新加载**
1. 在 `chrome://extensions/` 页面
2. 点击"移除"按钮
3. 点击"加载已解压的扩展程序"
4. 选择 `dist` 文件夹

**方法3：检查版本号**
1. 打开设置面板
2. 切换到"高级"标签
3. 查看"关于"部分
4. 版本应该显示：**0.4.0**

### 如果导出没反应

1. 检查浏览器的下载设置
2. 确保允许自动下载
3. 检查下载文件夹
4. 查看浏览器控制台是否有错误

### 如果导入失败

1. 确保文件是 JSON 格式
2. 确保文件是通过"导出所有数据"生成的
3. 检查文件内容是否完整
4. 查看错误提示信息
