# 快速开始指南 🚀

欢迎使用 AI Prompt 管理器！这个指南会帮你在 5 分钟内完成安装和使用。

## 📋 前置要求

- Node.js 18+ 
- Chrome 或基于 Chromium 的浏览器（Edge, Brave, Arc 等）
- 基础的命令行操作知识

## 🔧 安装步骤

### 步骤 1: 安装依赖

打开终端，进入项目目录：

```bash
cd prompt_manager
npm install
```

等待依赖安装完成（大约 1-2 分钟）。

### 步骤 2: 构建项目

```bash
npm run build
```

构建完成后，会在项目根目录生成 `dist` 文件夹。

### 步骤 3: 加载到浏览器

1. 打开 Chrome 浏览器
2. 在地址栏输入: `chrome://extensions/`
3. 在右上角开启「**开发者模式**」
4. 点击「**加载已解压的扩展程序**」
5. 选择项目中的 `dist` 文件夹
6. 完成！你会看到扩展已经安装

## 🎯 基础使用

### 方式一：右键保存 Prompt

1. 在任意网页选中一段文字（比如一个 AI 绘图提示词）
2. 右键点击 → 选择「**保存为 Prompt**」
3. 看到通知提示「Prompt 已保存」

### 方式二：打开管理器

**使用快捷键**（推荐）:
- Windows/Linux: `Ctrl + Shift + P`
- Mac: `Cmd + Shift + P`

**或者点击图标**:
- 点击浏览器工具栏的扩展图标
- 侧边栏会从右侧滑出

### 方式三：查看 Popup

- 点击扩展图标
- 会显示一个小窗口，包含统计信息和快速操作

## 💡 试试这些操作

### 1. 查看示例 Prompt

首次安装时，系统会自动添加 2 个示例 Prompt：
- 赛博朋克少女（包含多个标签和权重）
- 梦幻森林（简单的风格示例）

打开侧边栏查看它们！

### 2. 复制 Prompt

1. 在侧边栏找到任意 Prompt
2. 点击右下角的「复制」图标
3. 粘贴到你的 AI 绘图工具（Midjourney、Stable Diffusion 等）

### 3. 收藏重要的 Prompt

- 点击 Prompt 卡片右上角的星形图标
- 在排序选项中选择「收藏」，可以只看收藏的内容

### 4. 搜索 Prompt

- 在顶部搜索框输入关键词
- 支持搜索标题、描述、标签和 Prompt 内容

### 5. 删除不需要的 Prompt

- 点击卡片右下角的垃圾桶图标
- 确认后即可删除

## 🎨 理解标签系统

你会看到 Prompt 中有不同颜色的标签：

| 颜色 | 类型 | 说明 | 示例 |
|------|------|------|------|
| 🔵 蓝色 | 主体 (Subject) | 描述画面主体 | 1girl, castle |
| 🟣 紫色 | 风格 (Style) | 艺术风格 | cyberpunk, anime |
| 🟡 黄色 | 光效 (Lighting) | 光线效果 | neon lights, sunset |
| 🟢 绿色 | 构图 (Composition) | 画面构图 | close-up, wide angle |
| 🩷 粉色 | 技法 (Technique) | 质量控制 | masterpiece, 4k |
| ⚪ 灰色 | 自定义 (Custom) | 其他内容 | - |

标签旁的数字（如 `1.5`）是**权重**，表示这个元素在 AI 生成时的重要程度。

## 🐛 遇到问题？

### 扩展无法加载

- 确保运行了 `npm run build`
- 检查 `dist` 文件夹是否存在
- 尝试在 `chrome://extensions/` 点击「刷新」

### 右键菜单没有出现

- 重新加载扩展
- 刷新当前网页
- 检查是否选中了文字

### 侧边栏打不开

- 检查快捷键是否冲突（在 `chrome://extensions/shortcuts` 中修改）
- 尝试点击扩展图标打开

### 样式显示不正常

- 清空浏览器缓存
- 重新构建项目: `npm run build`
