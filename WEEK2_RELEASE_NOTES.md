# 🎉 v0.2.0 发布说明

## 标签编辑器 - 让 Prompt 编辑更轻松

我们非常高兴地宣布 AI Prompt 管理器 v0.2.0 正式发布！这个版本带来了期待已久的**可视化标签编辑器**。

---

## ✨ 主要新功能

### 1. 可视化编辑器 🎨

不再需要手动编辑复杂的数据结构，现在你可以：

- 📝 **直观编辑** - 可视化界面，所见即所得
- 🎯 **拖拽排序** - 拖动标签调整顺序
- ⚖️ **权重滑块** - 滑动调节每个标签的重要性
- 👀 **实时预览** - 即时查看生成的 Prompt

### 2. 标签库 📚

为你准备了 40+ 精选预设标签：

- 🎭 **6大分类** - 主体、风格、光效、构图、技法、自定义
- 🔍 **智能搜索** - 快速找到需要的标签
- 🌏 **中英对照** - 包含中文翻译，易于理解
- ➕ **一键添加** - 点击即可添加到编辑器

### 3. 完整的创建/编辑流程 ✏️

- ✅ **新建 Prompt** - 从零开始创建
- ✅ **编辑现有** - 修改任意已保存的 Prompt
- ✅ **正负面分离** - 分别管理正面和负面提示词
- ✅ **多格式支持** - Midjourney 和 Stable Diffusion

## 🎬 使用演示

### 新建一个 Prompt

1. 点击侧边栏右上角的"**新建**"按钮
2. 输入标题，例如："赛博朋克城市夜景"
3. 从右侧标签库选择预设标签，或点击"添加标签"创建自定义标签
4. 拖拽标签调整顺序
5. 使用滑块调整每个标签的权重
6. 在底部预览生成的 Prompt
7. 点击"保存"

### 编辑现有 Prompt

1. 在 Prompt 卡片上点击"编辑"图标
2. 修改标题、标签、权重
3. 保存更新

就是这么简单！

- ✅ 拖拽排序，直观高效
- ✅ 40+ 预设标签，开箱即用
- ✅ 滑块调节，实时反馈

