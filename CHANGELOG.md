## [0.2.0] - 2026-01-21

### 新增
- ✨ **可视化标签编辑器** - 完整的 Prompt 编辑界面
- ✨ **拖拽排序** - 使用 @dnd-kit 实现标签拖拽
- ✨ **权重滑块** - 直观的权重调节（0.5-2.0）
- ✨ **标签库** - 40+ 预设标签，分类浏览
- ✨ **实时预览** - 支持 Midjourney 和 Stable Diffusion 格式
- ✨ **新建功能** - 从零创建 Prompt
- ✨ **编辑功能** - 修改现有 Prompt

### 改进
- 🎨 更新侧边栏，集成编辑器模态框
- ⚡ 优化用户体验，操作更流畅
- 📝 添加中文翻译到预设标签

### 技术
- 📦 新增依赖: @dnd-kit/core, @dnd-kit/sortable, @dnd-kit/utilities
- 💻 新增组件: TagEditor, TagLibrary, PromptEditor
- 🔧 TypeScript 严格模式通过

## [0.1.0] - 2026-01-20

### 新增
- 🎉 首个 MVP 版本发布
- ✅ 右键菜单保存 Prompt
- ✅ 侧边栏管理界面
- ✅ Popup 快速访问窗口
- ✅ 一键复制功能（支持 Midjourney 和 Stable Diffusion 格式）
- ✅ 搜索和过滤功能
- ✅ 标签系统（6 种类型：主体、风格、光效、构图、技法、自定义）
- ✅ 权重管理（每个标签支持独立权重）
- ✅ 收藏功能
- ✅ 排序功能（按时间、收藏、标题）
- ✅ 示例数据自动加载
- ✅ 完整的 TypeScript 类型定义
- ✅ 本地存储（Chrome Storage API）

### 技术栈
- React 18 + TypeScript
- Vite 5 构建工具
- Tailwind CSS 3 样式框架
- Lucide React 图标库
- Chrome Extension Manifest V3

### 已知限制
- 暂不支持编辑功能（将在 v0.2.0 实现）
- 暂不支持图片上传
- 暂无数据导出功能
- 暂不支持云端同步

## 版本说明

### [0.1.0] - MVP 版本
第一周开发完成的最小可用产品（MVP），包含核心功能：
- 保存：右键保存选中文本
- 管理：侧边栏查看和管理 Prompt
- 复制：一键复制到剪贴板
- 搜索：快速查找 Prompt
- 标签：彩色标签和权重系统
